How To Install on Ubuntu
- 1. make folder
```bash
$ mkdir arduino
 ```
- 2. clone repository
```bash
 $ git clone http://github.com/agungdp22/arduino.git
 ```
- 3. make folder core of arduino
```bash
 $ make master_arduino
 ```
- 4. compile main/master.cc
```bash
 $ make
 ```
- 5. upload to arduino uno
```bash
 $ make master_upload
 ```
