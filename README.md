### `How To Install on Ubuntu`

- make folder
```bash
$ mkdir arduino
 ```
- clone repository
```bash
 $ git clone http://github.com/agungdp22/arduino.git
 ```
- make folder core of arduino
```bash
 $ make master_arduino
 ```
- compile main/master.cc
```bash
 $ make
 ```
- upload to arduino uno
```bash
 $ make master_upload
 ```
